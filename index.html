// ABOUT ME

Qui est Lemon ?

Je suis développeur autodidacte passionné par l'automatisation et la sécurité informatique sur Mac.

Ce qui me motive : construire mes propres outils plutôt que d'utiliser ce qui existe déjà. Chaque script résout un problème concret du quotidien.

Je m'intéresse à la sécurité réseau, à l'intelligence artificielle locale et à l'automatisation macOS.

Lemon Tools c'est mon laboratoire — construit outil par outil, appris ligne par ligne.

Un environnement complet

Chaque élément a été choisi et configuré pour s'intégrer parfaitement.

01

iTerm2 + Dracula

Terminal remplacé par iTerm2 avec thème Dracula, transparence et Nerd Fonts.

02

Oh My Zsh + Powerlevel10k

Prompt pro avec git, heure, statut des commandes et autosuggestions.

03

Outils modernes

eza, bat, fzf, zoxide — les commandes Unix remplacées par des versions 2024.

04

VS Code + Dracula

GitLens, Error Lens, Pylance, Live Server — stack Python + Web complète.

Launchd automation

3 services au démarrage — menubar, Ollama, network analyzer. Tout automatique.

06

Ollama + Mistral

LLM local, hors ligne, sans abonnement. Intégré dans le menu tools.
